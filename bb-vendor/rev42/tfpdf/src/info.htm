Description

Cette classe est une version modifiée de FPDF qui ajoute le support de l'UTF-8. De plus, elle incorpore dans le document uniquement les portions des polices qui sont effectivement utilisées, réduisant ainsi la taille du fichier PDF. Ces fonctionnalités ont été développées à l'origine pour le projet mPDF.

Pour utiliser une police Unicode dans votre script, passez le nom du fichier de police comme troisième paramètre à AddFont() et true comme quatrième paramètre. La police peut être située soit dans le répertoire font/unifont, soit directement dans le répertoire des polices systèmes (dans le cas où la constante _SYSTEM_TTFONTS a été définie). La famille de polices DejaVu est fournie dans l'archive.

Note : cette classe requiert l'extension mbstring.
